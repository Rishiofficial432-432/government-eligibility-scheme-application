# 🤖 AI Government Scheme Eligibility & Awareness Assistant

An AI-powered application that helps people discover government schemes they are eligible for based on personal details like age, income, occupation, and location. It includes an AI chatbot to simplify scheme-related queries.

## 📌 Project Status
🚧 **In development**  
The frontend UI is completed. Backend APIs, database integration, scheme recommendation logic, and the chatbot are in progress.

## 🎯 Problem Statement
Many people are unaware of government schemes due to complex eligibility criteria and lack of guidance.  
This system aims to:
- Provide personalized scheme recommendations
- Simplify information using AI
- Offer chatbot-based interaction

## ⚙️ How the Final System Will Work
1. User enters details in the UI
2. Backend processes input
3. Eligibility rules are applied
4. Database returns matching schemes
5. AI generates simplified explanations
6. Chatbot answers follow-up questions
7. Results are shown to the user

## 📊 Planned Database Schema
Table: `schemes`
- id
- name
- min_age
- max_age
- max_income
- gender
- occupation
- state
- category
- benefits
- documents
