## 📌 Project Overview
This project is an AI-based application that helps users find government schemes they are eligible for based on personal details like age, income, occupation, and location.

It also includes an AI chatbot to provide interactive assistance and simplify scheme-related queries.

---

## 🎯 Problem Statement
Many people are unaware of government schemes due to complex eligibility criteria and lack of proper guidance.

This system solves the problem by:
- Providing personalized scheme recommendations  
- Simplifying information using AI  
- Offering chatbot-based interaction

## ⚙️ How It Will Work (Final System)

1. User enters details in UI  
2. Backend processes input  
3. System checks eligibility using rules  
4. Database returns matching schemes  
5. AI generates explanations  
6. Chatbot answers queries  
7. Results shown to user
